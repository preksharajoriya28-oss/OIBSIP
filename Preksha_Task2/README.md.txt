📌 Task 2 – Logo Design (Elanique Aesthetic Studio)

 📖 Project Overview

This project is a *minimalist luxury logo design* created using *Canva* for a beauty and aesthetic brand named Elanique Aesthetic Studio . The logo reflects elegance, sophistication, and professionalism through a combination of modern serif typography, handwritten script, and a soft neutral color palette.

The design aims to establish a premium brand identity that resonates with clients looking for aesthetic, skincare, beauty, and self-care services. Every element of the logo has been carefully arranged to create a clean, memorable, and timeless visual identity.


🎯 Objective

* Create a unique and memorable brand identity.
* Reflect elegance, luxury, and professionalism.
* Build trust through a clean and modern logo.
* Represent beauty, skincare, and aesthetic services.
* Design a versatile logo suitable for both digital and print platforms.



✨ Features

* Elegant minimalist design.
* Premium serif typography.
* Handwritten script overlay for a personalized touch.
* Soft neutral color palette.
* Fashion illustration enhancing the beauty theme.
* Clean and balanced layout.
* Modern luxury branding style.



🎨 Design Process

The logo was designed using *Canva* by combining a sophisticated serif font with a handwritten script to create a luxurious and artistic appearance. A soft glowing light effect was added in the background to highlight the brand name without overpowering the design. Neutral beige and brown tones were selected to communicate warmth, elegance, and trust, while the fashion illustration was included to reinforce the brand's focus on beauty and aesthetics.



🛠️ Tools Used

* Canva
* Canva Typography
* Canva Elements
* Glow Effects
* Fashion Illustration

 🎨 Color Palette

* Beige
* Light Cream
* Dark Brown
* Soft White

These colors create a luxurious, calming, and premium visual identity suitable for the beauty industry.

🔤 Typography

The logo combines *modern serif typography with an elegant handwritten script*,creating a perfect balance between professionalism and creativity. Wide letter spacing in the tagline improves readability while maintaining a sophisticated appearance.

 🎨 Design Elements

* Elegant serif typography
* Handwritten signature-style text
* Soft glow background effect
* Fashion illustration representing beauty
* Minimalist composition
* Neutral luxury color palette

👥 Target Audience

* Beauty enthusiasts
* Skincare clients

* Women seeking aesthetic treatments
* Luxury beauty brands

🚀 Outcome

The final logo successfully establishes a luxury and elegant brand identity for Elanique Aesthetic Studio .Its clean typography, balanced composition, and premium color palette make it suitable for business cards, social media, websites, packaging, signage, and other branding materials. The design effectively communicates trust, sophistication, and modern beauty aesthetics.
