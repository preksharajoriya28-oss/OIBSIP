📌 Task 4 – Infographic Design (Software Development Lifecycle)

 📖 Project Overview

This project is an educational infographic created using Canva to visually represent the Software Development Lifecycle (SDLC). The infographic presents the complete software development process in a simple, organized, and engaging timeline format, making it easier for students, beginners, and professionals to understand each stage involved in software development.

The design follows a clean and colorful layout where every milestone is represented with a unique color, relevant icon, and concise description. The use of visual hierarchy, spacing, and modern design elements improves readability while making complex information easier to understand.

 🎯 Objective

* Explain the complete Software Development Lifecycle (SDLC) in a simple visual format.
* Improve understanding of each software development phase.
* Present technical information in an engaging and organized manner.
* Create an infographic suitable for educational and presentation purposes.
* Enhance information retention through visual learning.

 ✨ Features

* Clean and modern infographic layout.
* Timeline-based representation of SDLC phases.
* Color-coded milestones for easy understanding.
* Relevant icons for each development stage.
* Simple and readable typography.
* Well-balanced spacing and alignment.
* Educational and professional design.


🎨 Design Process

The infographic was designed using *Canva* by organizing the Software Development Lifecycle into seven clearly defined milestones. Each stage was assigned a different color and icon to improve visual distinction and maintain user engagement. A horizontal timeline was added at the bottom to represent the sequential flow of the software development process.

The layout focuses on simplicity and readability by maintaining consistent spacing, typography, and alignment throughout the design. The combination of pastel colors and minimal design elements creates a professional yet attractive educational infographic.
